# Changelog

## 2026-04-14（下午）

- **前端全站响应式适配**（home.html / score.html / chat.html）：
  - PC 端采用黄金分割比例（38.2% 视觉焦点偏移）居中布局。
  - 移动端（<640px）自动单列化、间距与字号收敛，保证移动端易读易用。
- **AI 智能问答正式接入**：
  - 前端问答页（chat.html）完整接入后端 `POST /api/qa/ask` 接口。
  - 后端检索 `docs/faq.txt` 知识库内容，结合 DeepSeek AI 生成回答。
  - AI 回答在前端以气泡形式渲染展示。

## 2026-04-14（上午）

- 优化成绩查询前端页面（UI 统一、结果表格展示、加载态与错误提示）。
- 新增智能问答前端页面：`static/chat.html`（回车发送问题）。
  - 空状态：浅色蓝紫渐变背景 + 输入框周围轻量动态渐变光晕。
  - 聊天状态：输入框固定底部、上方展示聊天记录。
- 新增入口前端页面：`static/home.html`（跳转成绩查询/问答系统）。
  - 根路径 `/` 与 `HEAD /` 跳转到入口页。
- 新增问答 JSON API：`POST /api/qa/ask`。
- 增加前端输入有效性校验：
  - 姓名仅允许中文汉字输入（兼容中文输入法）。
  - 学号仅允许数字且必须为 11 位。
  - 校验失败时在输入框下方显示红色提示：输入格式错误，请重新输入。
- 修复 SQLite 数据库目录不存在导致的启动失败：自动创建数据库文件所在目录。
- 修复数据库初始化脚本在直接运行时的导入问题：支持 `python3 scripts/init_db.py` 执行。
